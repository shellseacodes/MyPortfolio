Sorting an Itemized List in Ascending Order

In a team, we developed a versatile program that simplifies the task of managing and searching for items. The program allows users to store items in a text file and provides an intuitive prompt menu for various operations. Users can seamlessly search for items by their unique ID, price, or name, with the results conveniently displayed in ascending order. This user-friendly solution streamlines item management and retrieval, making it a valuable tool for efficient data handling and organization.
